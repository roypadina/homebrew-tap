# roypadina/homebrew-tap

Homebrew tap for [@roypadina](https://github.com/roypadina)'s macOS apps.

## Install

```bash
brew install --cask roypadina/tap/languard
```

(or `brew tap roypadina/tap` then `brew install --cask languard`)

## Casks

| Cask | Description |
|---|---|
| **languard** | Menu-bar app: turns Wi-Fi off on wired LAN, back on when unplugged. [Repo](https://github.com/roypadina/LanGuard) |
